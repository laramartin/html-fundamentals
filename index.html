HTML drives the web. Whether you use IE, Chrome, Firefox, Safari, or some other browser, many of us use the web daily. In this course you will learn the fundamental elements of creating web pages using HTML.

Not covered in this course is detailed information about CSS, javascript, or HTTP as those topics are covered in detail in other Pluralsight courses 1, 2 .

Objectives

  • Explain the fundamental concepts of HTML
  • Have fun
    • creating documents
    • creating memories
  • Mark it up!
